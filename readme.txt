1. Формируем отчет товароведа по необходимым группам или сегменту номенклатуры.
2. В настройках отчета должна быть установлена галочка Отображать ДМО.
3. Остальные галочки должны быть сняты, включая галочку Убрать пустый строки.
4. Созраняем отчёт товароведа в файл, в папку "Исходные данные". 
В наименовании файла обязательно должно содержать сочетание букв "ОТ".
5. Формируем Минимальные остатки с отобором тех же папком или сегмента, что и Отчет товароведа. 
Выбираем только один из МО внешний или внутренний в заивисимости от того, какое МО необходимо проанализировать.
6. Сохраняем Минимальные остатки в файл, в папку "Исходные данные".
В наименовании файла обязательно должно содержать сочетание букв "МО".
7. Запускаем файл "Расчёт ДМОср по подразделениям.exe"
8. Результат расчета ДМОср и установленный МО для переноса сохраняется в файл "Анализ установленного МО.xlsx"
9. Далее Заказчик анализрует данные в колонке "МО для переноса" и при необходимости вносит корректировки вручную.
10. После окончания корректировки, файл сохраняем и запускаем следующую программу для подготовки файла для загрузки МО в 1С.
